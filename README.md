# 微信支付 Agent Skills

Agent 需要指令来引导。指令越好，Agent 就越能帮开发者高效完成微信支付的接入工作。

微信支付提供了：

- **LLM 可读的文档** — 结构化的业务知识、接入指南和排障手册，Agent 可直接检索理解
- **代码示例库** — API 调用示例（含签名），Agent 检索后展示，开发者复制即用
- **Agent Skills** — 引导 Agent 在特定产品领域正确帮助开发者的技能文件

本目录是微信支付 Agent Skills 的合集。每个 Skill 对应一个微信支付产品，定义了 Agent 的能力范围、交互规范和引用资料索引。
