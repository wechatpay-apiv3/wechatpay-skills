# 什么是品牌API证书？如何获取品牌API证书和私钥？

## 功能介绍

品牌API证书，是指由品牌商户申请的，用来证实品牌商户身份的证书。API证书由证书授权机构 Certificate Authority（简称 CA）颁发。证书中包含品牌商户的品牌ID、公司名称、公钥等信息。请妥善保管API证书文件，防止泄露。

## 使用场景

品牌API中，调用微信支付所有接口的签名和敏感字段信息解密需要用到品牌API证书。

## 如何获取品牌API证书

获取品牌API证书的操作步骤详见品牌经营平台的"下载并配置品牌API证书"文档。

## 如何查看品牌API证书序列号？

登录[品牌经营平台](https://pay.weixin.qq.com/xdc/brandhomeweb/brand/home)【账号管理】->【安全中心】->【品牌API证书】，可查看品牌API证书序列号。

也可以使用第三方的证书解析工具查看证书内容，或者使用 openssl 命令行工具查看证书序列号：

```bash
openssl x509 -in 1900009191_20180326_cert.pem -noout -serial
# serial=1DDE55AD98ED71D6EDD4A4A16996DE7B47773A8C
```

## 如何获取品牌API证书私钥？

申请品牌API证书时，证书工具会生成品牌API证书私钥，并保存在本地证书文件夹的文件 `apiclient_key.pem` 中。私钥也可通过工具从 p12 证书中导出。请妥善保管好品牌API证书私钥文件。

> **提示**：请勿将私钥文件暴露在公共场合，如上传至 GitHub 或将其写入客户端代码等。
