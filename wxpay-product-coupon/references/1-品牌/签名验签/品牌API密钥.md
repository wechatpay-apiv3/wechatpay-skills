# 什么是品牌API密钥？如何获取品牌API密钥

## 1、功能介绍

微信支付品牌API处理回调通知时，为防止报文被其他人恶意篡改，服务器会对数据进行加密。品牌商户收到报文后，要解密出明文，解密过程中用的 key 就是品牌API密钥。

> **注意**：品牌API密钥属于敏感信息，请妥善保管不要泄露，如果怀疑信息泄露，请重设密钥。

## 2、设置要求

1. 请与贵司技术人员沟通设置品牌API密钥（密钥为32个字符，支持数字和大小写字母组合）
2. 确保技术人员保管品牌API密钥，防止泄露
3. 将技术人员设置为"安全联系人"，同步接收相关的风险提醒

## 3、如何设置品牌API密钥

详细步骤请参考品牌经营平台的"配置品牌API密钥"操作指引。

## 4、如何更新品牌API密钥

### 4.1 更换的注意事项

1. 品牌管理员与所有相关服务的技术沟通，明确更换品牌API密钥的方案、时间点，建议选择没有或较少交易的时间段。修改品牌API密钥后，旧密钥会**立即失效**，若线上仍存在使用旧密钥的服务，会导致API调用失败。
2. 修改密钥之前，建议暂停相关业务服务，例如发出系统维护公告。
3. 按照操作完成更换后恢复业务服务使用。

### 4.2 更换的操作说明

1. 管理员登录【微信支付品牌经营平台 -> 账户管理 -> 安全中心】，找到品牌API密钥，点击"修改"
2. 输入需要修改的新密钥，确认技术人员已制定更换密钥计划，并将新密钥已妥善保管。（注意：密钥设置成功后不支持查看，仅支持修改重新设置），确认无误后点击"确认修改"
3. 安全验证：按提示输入操作密码 + 手机验证码，提交确认即可修改成功
4. 品牌API密钥修改成功，请联系贵司技术人员使用新密钥
