# 配置APIv3密钥

## 1、功能介绍

微信支付APIv3的下载平台证书接口以及处理回调通知中，为防止报文被其他人恶意篡改，服务器会对数据进行加密。商户收到报文后，要解密出明文，解密过程中用的 key 就是 APIv3 密钥。

> **注意**：APIv3 密钥属于敏感信息，请妥善保管不要泄露，如果怀疑信息泄露，请重设密钥。

## 2、使用说明

1. APIv3 密钥的设置与修改不影响 APIv2 密钥
2. APIv3 密钥与 APIv2 密钥是隔离的，设置该密钥时不会导致 APIv2 密钥发生变化
3. APIv3 密钥主要用于平台证书解密、回调信息解密

## 3、设置要求

1. 请与贵司技术人员沟通设置的 APIv3 密钥（密钥为32个字符，支持数字和大小写字母组合）
2. 确保技术人员保管 APIv3 密钥，防止泄露
3. 将技术人员设置为"技术负责人账号"，同步接收相关的风险提醒

## 4、设置APIv3密钥

超级管理员登录微信商户平台，进入【账户中心 > 账户设置 > API安全】目录，点击"设置APIv3密钥"。

操作步骤：
1. 登录商户平台，进入【账户中心 - API安全 - 解密回调 - APIv3密钥】点击"设置"
2. 确认弹出窗口的内容已和技术人员沟通好后点击"已沟通"
3. 输入需要设置的 APIv3 密钥，点击"确定"（密钥为32个字符，支持数字和大小写字母组合。设置成功后不支持查看，只支持重新设置）
4. 安全验证，输入操作密码、短信验证码，点击"确定"即可设置成功
5. 点击"完成"，APIv3 密钥设置成功，请联系技术人员使用新密钥

## 5、修改APIv3密钥

### 注意事项

1. 修改前请与贵司内部技术人员沟通是否明确需要修改，避免未沟通直接修改而影响业务使用
2. 修改密钥之前，建议暂停相关业务服务，例如发出系统维护公告
3. 技术人员在所有相关服务中更换为新的 APIv3 密钥，并测试验证服务可用

### 操作说明

超级管理员登录【微信支付商户平台 -> 账户中心 -> 账户设置 -> API安全 -> APIv3密钥 -> 修改】

1. 登录商户平台，进入【账户中心 - API安全 - 解密回调 - APIv3密钥】点击"修改"
2. 阅读弹框中事项，若已沟通点击"已沟通"
3. 输入需要设置的 APIv3 密钥，点击"确定"（密钥为32个字符，设置成功后不支持查看，只支持重新设置）
4. 安全验证，按提示输入手机验证码，点击"确定"即可修改成功
5. 点击"完成"，APIv3 密钥修改成功，请联系技术人员使用修改后的新密钥
